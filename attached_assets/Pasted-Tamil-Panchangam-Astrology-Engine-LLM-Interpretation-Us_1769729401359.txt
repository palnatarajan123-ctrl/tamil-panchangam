💾 PART 7 — PERSISTENCE & CACHING

CREATE TABLE IF NOT EXISTS prediction_llm_interpretation (
    id UUID,
    base_chart_id TEXT,
    period_type TEXT,
    period_key TEXT,
    feature_name TEXT,
    provider TEXT,
    model TEXT,
    prompt_version TEXT,
    prompt_tokens INTEGER,

    created_at TIMESTAMP

CREATE TABLE IF NOT EXISTS llm_token_usage (
    id UUID,
    feature_name TEXT,
    prompt_version TEXT,
    total_tokens INTEGER,
    created_at TIMESTAMP
);
Reuse cached interpretation if:
(base_chart_id, period_type, period_key, feature_name, prompt_version)
matches exactly.
 
💰 PART 8 — MONTHLY TOKEN BUDGET (ENGINE-LEVEL)
Config:
LLM_MONTHLY_TOKEN_BUDGET = 50_000
Budget Query:
SELECT COALESCE(SUM(total_tokens), 0) AS tokens_used
FROM llm_token_usage
WHERE DATE_TRUNC('month', created_at) = DATE_TRUNC('month', CURRENT_DATE);
If budget exceeded → deterministic fallback.
